# 💳 Credit Card Fraud Detection Using Machine Learning

## 📌 Project Overview

This project presents an end-to-end machine learning pipeline for detecting fraudulent credit card transactions. Due to the highly imbalanced nature of the dataset, multiple evaluation metrics were used to compare model performance.

---

## 🎯 Objectives

- Perform Exploratory Data Analysis (EDA)
- Handle class imbalance
- Train multiple machine learning models
- Compare model performance
- Generate business insights

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📂 Dataset

- Total Transactions: **284,807**
- Fraudulent Transactions: **492**
- Genuine Transactions: **284,315**

Dataset Source:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

---

## 🤖 Models Used

- Logistic Regression
- Random Forest Classifier

---

## 📊 Results

| Metric | Logistic Regression | Random Forest |
|--------|--------------------:|--------------:|
| Accuracy | 99.919% | **99.960%** |
| Precision | 82.50% | **94.12%** |
| Recall | 67.35% | **81.63%** |
| F1 Score | 74.16% | **87.43%** |
| ROC-AUC | 95.01% | **96.30%** |

---

## 💼 Business Impact

The Random Forest model demonstrated superior fraud detection capability while maintaining high precision and recall, making it a strong candidate for real-world fraud detection systems.
